# Beta Features

**Experimental. Subject to change. Use at your own risk.**

These features are under active development. APIs may break between releases.

## Browser Inspector

Debug and analyse web pages with structured captures.

| Feature | Description | Status |
|---------|-------------|--------|
| [ot-browse](ot-browse.md) | Browser debugging TUI | Beta |
| [Browser Inspector](browser-inspector.md) | Page inspection tools | Beta |
| [Page View Tools](page-view.md) | Analyse browser captures | Beta |
